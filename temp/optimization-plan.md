# 贪吃蛇游戏优化计划

创建时间：2026-03-20 08:35

## 目标
修复用户反馈的三个问题并自行评估其他优化项

## 问题清单

### 1. 背景音乐缺失 🔴
**原因分析：**
- `initBGM()` 在 `startGameFromMenu()` 中调用
- 但音频上下文可能没有正确初始化
- 和弦播放逻辑可能有问题

**修复方案：**
- 确保 `initAudio()` 在 `initBGM()` 之前调用
- 检查和弦播放逻辑
- 添加调试日志确认 BGM 状态

### 2. 道具倒计时闪烁 🔴
**原因分析：**
- `updatePowerUpIndicator()` 在每次 `update()` 中被调用（每帧）
- DOM 元素被频繁重绘导致闪烁

**修复方案：**
- 只在道具状态实际变化时更新 UI
- 使用 `setInterval` 每秒更新一次倒计时显示
- 避免在 `gameStep` 中调用 UI 更新

### 3. 其他优化项 🟡
**评估发现：**
- ✅ 代码结构：单文件 HTML，逻辑混杂
- ✅ 性能：粒子系统没有上限，可能导致内存增长
- ✅ 用户体验：没有 mute 按钮快捷键
- ✅ 视觉效果：可以增加更多动画效果
- ✅ 音效：BGM 循环可能不流畅

**优化方案：**
1. 添加粒子数量上限
2. 优化 BGM 循环逻辑
3. 添加键盘快捷键（M 静音）
4. 改进道具消失视觉效果
