# 🎬 优量汇广告接入指南

## ⚠️ 重要：上线前必须修改的配置

### 1. 注册腾讯优量汇开发者账号

1. 访问：https://e.qq.com/
2. 注册账号（需要手机号）
3. 完成实名认证（身份证）
4. 创建应用（选择"移动应用" → "Android"）
5. 创建广告位（选择"激励视频"类型）

### 2. 获取配置信息

创建应用后，你会获得：
- **App ID**：应用唯一标识（数字）
- **Ad Slot ID**：广告位 ID

### 3. 修改配置

编辑 `app/src/main/java/com/snake/game/MainActivity.java`：

```java
// 第 22 行：替换为你的 App ID
private int appId = "1109652815"; // 改为你的 App ID

// 第 21 行：替换为你的广告位 ID
private String adSlotId = "887249b0"; // 改为你的 Ad Slot ID
```

### 4. 测试设备 ID

在优量汇后台，将你的设备 ID 加入白名单，否则会看不到广告。

### 5. 收益提现

- 满 100 元可提现
- 每月 15 日结算上月收益
- 打款到银行卡或支付宝

## ⚠️ 注意事项

1. **不要诱导点击**：违反政策会被封号
2. **广告频率控制**：同一用户两次广告间隔至少 30 秒
3. **如实申报收入**：个人年收入超 12 万需要报税
4. **遵守平台规则**：TapTap 对广告位置有要求
